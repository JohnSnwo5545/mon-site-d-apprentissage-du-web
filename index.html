Concepts clés

Partie 1 - Découvrez le web et ses origines

Web

correspond à l'ensemble des interactions entre les pages d'un site web .Notamment avec W3C le premier site web crée par Tim Berners-Lee.le W3C a été crée dans le labo de recherche CERN

Internet

Anciennement "Arpanet" un réseau militaire décentralisé,qui maintenant s'appelle internet.L'internet permet la communication entre les ordinateurs

Cloud

le cloud correspond au web plus précisément l'intermédiaire entre les services proposé par le web et entre l'utilisateur

W3C

W3C signifie World wide web consortium crée par Tim Berners-Lee au cern , c'est une organisation qui s'occupe de mettre à jour les nouvelles fonctionnalités des techologies d'aujourd'hui comme les languages de programmation notamment le HTML

Partie 2 - Distinguez les langages du Web

Langages client

le language client correspond à ce que l'utilisateur pourra modifier et observer , c'est ce qu'on n'appelle la partie front-end avec le HTML, CSS, Javascript.

Navigateurs

le navigateur permet à l'utilisateur de rechercher un site web grâce avec son url. comme Google chrome , Safari.com.

Langages serveur

Le langage serveur fonctionne avec un langage client qui correspond à la machine(ordinateur),la première étape est que le langage client transmetla demande de l'utilisateur vers un serveur qui stocke les données d'un site web comme openclassroom,deuxième étape correspond à la génération du site avec un langage comme PHP , python, ruby ou encore java puis renvoit le site en code HTML , CSS et Javascript compréhensible à l'utilisateur

Frameworks

c'est des fonctionnalités qui permettent aux développeurs de coder de manière plus rapide et lui faciliter le travail qui serai beaucoup trop fastidieux et prendrait trop de temps notamment Django pour python, Symphony pour php.

Bases de données

une base de donnée est une archive qui contient des données en quantité énorme .IL existe des logiciels de base de donnée comme My SQL,PosterSQL,Oracle.ON utilise le language "SQL pour communiquer avec ces logiciels "

Site responsive

un site responsive est capable de s'adapter en la taille d'un écran(ordinateur et tablette ou téléphone).

Application mobile

on n'utilise des logiciels pour crée des applications mobiles qui consiste à coder pour un système d'exploitation précis comme IOS et android. Il est nécessaire de maîtriser des langages comme Kotlin, java ou swift.

Partie 3 - Découvrez le fonctionnement du réseau

Serveurs

les serveurs sont stockés dans un data center dans le monde entier.Les serveurs permettent la transmission d'information entre différents serveurs comme un serveur se trouvant à paris et un autre aux Etats-unis. Ses données sont transmis par des canals sous marins , ce qui rend plus rapide la transmission de données et donc une recherche plus fluide par l'utilisateur car l'informations part de serveur en serveur en quelques milisecondes.

Adresse IP

l'adresse IP permet de retrouver des serveurs de site web comme google , l'IP est attribuer à chaque site qui peut avoir plusieurs serveurs à la fois et donc un IP multiple ou qui peut changer.

Nom d’hôte et DNS

ses deux informations permettent de trouver un ordinateur sur le web.Pour cela il existe le nom d'hôte qui facilite la recherche d'un site par exemple google.com , le nom d'hôte correspond à une adresse IP qui est attribuer comme 123.34.56 c'est l'annuaire DNS qui permet de convertir l'IP en un non d'hôte plus facile à utiliser sur le web.

protocoles

des protocoles comme https: , TCP ou d'autre ont été crée pour permettre aux ordinateurs de communiquer entre eux. ils existent des protocoles de bas niveau comme le TCP(transmission control protocol) et les hauts niveaux comme HTTP ou HTTPS(Hyper Text Transfer protocole,HTTP securisé)

Partie 4 - Identifiez les métiers du web

Développeurs

Ils existent des développeurs Front-end qui travaillent avec la partie visible du site et les développeurs back-end qui travaillent avec la partie non-visible comme les serveurs.

Product Managers et chefs de projets

Le product Managers et le chef de projets collaborent ensemble pour finaliser un produit et le mettre sur le marché.Mais n'ont pas les mêmes missions pourtant le même objectif réaliser un produit qui conviendra aux utilisateurs futurs et les clients.Le product manager définit les indicateurs clés de succès du produit il comprend les besoins des consommateurs et les transmet à l'équipe.Alors que le chef de projet collabore et communique avec les parties prenantes et la direction , il crée et maintient des processus pour les changements dans le projet.

Designers

Le designers s'occupe de la partie front-end du produit donc de son apparence , c'est lui qui définit le style et à quoi doit ressembler le produit , il corrige et ajuste en s'adaptant à la demande du client et des futurs consommateurs , il dirige donc une partie de l'équipe au niveau de toutes l'apparence et le design principal du produit.

Testeurs

le testeur est celui qui teste les sites webs avant leur publication pour valider un projet s'il correspond ou non à la demande des utilisateurs.

Techniciens informatique

Il s'occupe de la partie technique des ordinateurs pour qu'ils puissent fonctionnés et permettrent à l'équipe des développeurs de travailler efficacement avec le matériel à disposition, il répond aux questions techniques des clients ou des développeurs sur les ordinateurs ou la technologies .

Langages de développement web

HTML

Front-End

Squelette du site Web. Il définit la structure de base du contenu, comme les titres, les paragraphes, les images, etc.

CSS

Front-End

l'esthétique du site web. IL permet de définir à quoi vas ressembler le site web, comme la couleur des titres , les types de polices la taille des charactères...

JavaScript

Front-End

Il permet de faire fonctionner correctement le site en implimentant des fonctionnalités pour dynamiser le site web ,cela grâce à des framework comme notamment react pour javascript

JavaScript

Back-End

Le back-end correspond aux serveurs , notamment la demande de créer un compte et d'autre fonctionnalité permettant de rendre le site plus efficace et fluide lors du résultat finale.

PHP

Back-End

le PHP est l'un des languages utilisés par les développeurs back-end , et c'est un language côté serveur comme python ou d'autre.

Java

Back-End

c'est un language de programmation de haut niveau orientée objet crée par James Gosling et Patrick Naughton, il est utilisé dans le développement d'application web et serveurs. Notamment ce language excelle dans l'optimisation des ressources serveurs.

Python

Back-End

le language python est un des languages serveurs le plus utilisé , il est facile à apprendre et s'adapte bien à tout projet il est utilisé dans la data science et dans la robotique.

C#

Back-End

Le C# est utilisé dans la création de logiciel et de jeu vidéo . Il a été conçu par Microsoft.IL est basée sur une technologie orientée objet(POO), il est utile dans le typage fort et statique(int,string,bool),et pour faire du générique c'est à dire écrire du code flexible destiné à plusieurs types en gardant la sécurité offerterte par le typage.

SQL

Bases de données

une base de donnée est une archive contenant une immensité de données , elle est très utile dans la création de fonctionnalités puissantes comme stocker des informations clients ou des formulaires automatisée, pour rappel les base de données en un languages spécifiques le SQL(Oracle,Postersql,Mysql)

  • IDE comme VS Code : integrated development environnement qui permet d'écrire et d'afficher du code dans un terminal ou dans un serveur locale
  • Github : un service d'hébergement et de gestion de logiciels, il utilise le,logiciel git
  • Librairies / bibliothèques : les bibliothèques correspondent à des bouts de code crée par d'autre dévloppeurs qui sont mis à disposition pour être utiliser dans des projets.on peut les trouver dans github ou les télécharger dans l'IDE
  • Frameworks : un Frameworks sont des fonctionnalités qui aide les développeurs à travailler plus rapidement , notamment pour des fonctionnalités souvent utilisé dans le javascript comme le react , pour python c'est django et Symfony pour PHP
